## 📖 Sobre o Projeto

Este repositório contém o código-fonte de um protótipo web desenvolvido especificamente como parte de um **desafio técnico para uma entrevista**. 

O projeto foi inspirado na identidade visual da **FURIA** (uma das maiores organizações de esports do mundo), com o objetivo de demonstrar minhas habilidades na construção de interfaces modernas, responsivas e alinhadas com o universo gamer e competitivo.

🔗 **[Acesse o protótipo rodando ao vivo clicando aqui!](https://davifariass.github.io/prot-tipo-FURIA/)**

---

## 🎯 Objetivo e Funcionalidades

O foco principal deste protótipo foi criar uma experiência de usuário (UX) fluida e um design (UI) de alto impacto, características essenciais no cenário de esports.

- [x] **Design Responsivo:** A interface se adapta perfeitamente a diferentes tamanhos de tela (Mobile, Tablet e Desktop).
- [x] **Identidade Visual:** Uso da paleta de cores, tipografia e elementos visuais que remetem à agressividade e profissionalismo da marca FURIA.
- [x] **Navegação Intuitiva:** Estruturação clara das informações para facilitar a jornada do usuário dentro da landing page.

---

## ⚙️ Tecnologias Utilizadas

A construção da interface foi feita utilizando as seguintes tecnologias e ferramentas:

*   **Front-end:** `HTML5`, `CSS3` e `JavaScript`
*   **Hospedagem:** `GitHub Pages` (Para o deploy rápido e visualização do protótipo)
